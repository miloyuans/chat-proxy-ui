Authorize ChatGPT Pro

推荐方式：手动注入 Cookie（最稳定）

操作步骤：
1. 用 Chrome 浏览器打开 chat.openai.com
2. 登录你的 ChatGPT Pro 账号（确保能正常聊天）
3. 安装 Chrome 扩展：EditThisCookie 或 Cookie-Editor
4. 在 chat.openai.com 页面，点击扩展图标 → Export → 选择 JSON 格式
5. 将导出的 JSON 完整粘贴到下方文本框（不要修改格式）
6. 点击“注入 Cookie 并授权”按钮

提示： Cookie 通常有效期 7–30 天，过期后需重新登录并导出新的 Cookie 重新授权。

注：本项目仅用于内部测试与学习，遵守 OpenAI 使用条款，避免高频滥用。
